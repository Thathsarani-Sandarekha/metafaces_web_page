MetaFaces: Consistent Multi-Character Image Generation

Thathsarani Sandarekha1, Prasan Yapa1
1Robert Gordon University,
Paper (Coming Soon) arXiv (Coming Soon) Code (Coming Soon)

Abstract

Generating consistent and coherent character representations across multiple images is a critical challenge in visual storytelling, particularly in diffusion-based generative models.

Existing methods often struggle to maintain characterspecific attributes and spatial consistency when transitioning between scenes. In this paper, we propose MetaFaces, a novel framework leveraging dynamic memory networks and anchordriven attention mechanisms to address this limitation. MetaFaces introduces an adaptive memory module to store and retrieve character-specific data, including anchor images and attention maps, enabling persistent representation throughout the diffusion process. By dynamically integrating attention masks into a modified UNet architecture, our approach ensures consistency in character attributes while preserving generative diversity.

Additionally, MetaFaces employs advanced grounding and segmentation techniques using Grounding DINO and SAM to refine anchor masks for enhanced spatial coherence. Experimental results demonstrate significant improvements, with MetaFaces achieving the highest CLIP Score (0.3619) and DreamSim Score (0.1753) compared to state-of-the-art methods. This work establishes a new paradigm for consistent multi-image generation, with applications in storytelling, comics, and visual narrative creation.
